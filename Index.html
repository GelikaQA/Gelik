What is Selenium IDE?

Selenium IDE

Selenium IDE is a part of the Selenium suite and was developed to speed up the creation of automation scripts. It's a rapid prototyping tool and can be used by engineers with no programming knowledge.

Key points:

  • Record & replay tool
  • Not require any programming knowledge
  • Initially was a Firefox add-on
  • Easy-to-use interface to build automation
